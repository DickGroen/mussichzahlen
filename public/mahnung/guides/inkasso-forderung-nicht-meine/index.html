Warum kommen Forderungen für Verträge die man nie abgeschlossen hat?

Es gibt mehrere mögliche Ursachen: Datenverwechslung beim Inkassounternehmen, Identitätsmissbrauch durch Dritte, ein Vertrag der ohne Ihr Wissen im Internet abgeschlossen wurde, oder eine Forderung die an den falschen Empfänger zugestellt wurde. In allen Fällen gilt: Sie sind nicht verpflichtet zu zahlen solange die Vertragsgrundlage nicht nachgewiesen ist.

Was Sie sofort tun sollten

Schriftlich Widerspruch einlegen — erklären Sie dass Sie die Forderung nicht kennen und fordern Sie den vollständigen Nachweis der Vertragsgrundlage an. Per Einschreiben mit Rückschein.

Unterlagen prüfen — prüfen Sie Ihre eigenen Unterlagen auf Verträge mit dem genannten Gläubiger. Schauen Sie auch in alten E-Mails nach.

Schufa-Selbstauskunft anfordern — prüfen Sie ob unter Ihrem Namen bereits weitere unbekannte Forderungen bestehen.

Bei Identitätsmissbrauch: Anzeige erstatten — wenn Sie vermuten dass jemand unter Ihrem Namen Verträge abgeschlossen hat erstatten Sie Anzeige bei der Polizei. Das Aktenzeichen hilft beim Bestreiten der Forderung.

Was das Inkassounternehmen nachweisen muss

Das Inkassounternehmen ist in der Beweispflicht — nicht Sie. Es muss auf Anfrage nachweisen dass ein gültiger Vertrag mit Ihnen besteht, dass die Forderung berechtigt ist, und dass das Unternehmen berechtigt ist diese Forderung einzutreiben. Können sie das nicht liefern haben Sie starke Argumente die Forderung vollständig abzulehnen.

Nicht zahlen um die Sache zu beenden — eine Zahlung kann als Anerkennung der Forderung gewertet werden und ermutigt weitere ähnliche Forderungen.

Nicht ignorieren, wenn ein Mahnbescheid kommt — ein Mahnbescheid vom Amtsgericht erfordert innerhalb von 14 Tagen eine Reaktion.

Häufige Fragen

Was wenn das Inkassounternehmen trotzdem weitermacht?

Wenn Sie schriftlich widersprochen haben und die Forderung dennoch weiterverfolgt wird können Sie sich an die Verbraucherzentrale oder einen Anwalt wenden. Bei offensichtlichem Betrug wenden Sie sich an die Polizei und die Bundesnetzagentur.

Kann die Forderung trotzdem in meine Schufa eingetragen werden?

Ein unberechtigter Eintrag kann nachträglich gelöscht werden. Wenn Sie einen Betrugsfall vermuten informieren Sie auch die Schufa direkt.

Wann sollte ich einen Brief hochladen?

Wenn Sie unsicher sind, ob die Forderung Sie überhaupt betrifft oder wie Sie sie abwehren. Sie laden das Schreiben hoch und erhalten eine kostenlose Ersteinschätzung — bevor Sie zahlen oder antworten. So entscheiden Sie in Ruhe und auf einer klareren Grundlage.

Unsicher, ob diese Forderung Sie betrifft?

Sie müssen nicht sofort entscheiden. Laden Sie das Schreiben zuerst hoch und erhalten Sie eine kostenlose Ersteinschätzung — worum es geht und wie Sie eine unbekannte Forderung abwehren können. Auf Wunsch folgt die vollständige Analyse mit fertigem Antwortschreiben.
